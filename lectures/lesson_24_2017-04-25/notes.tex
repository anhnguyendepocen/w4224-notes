We can use the built-in R function \tt{acf} to calculate the effective sample size calculation.

\begin{lstlisting}
a.sim <- thetachain[1]
b.sim <- thetachain[2]
acf(a.sim)

rho.a <- acf(a.sim)$acf
rho.a <- rho.a[-1]

chain.length / (1 + 2 * sum(rho.a))
\end{lstlisting}

We should get an effective sample size of 300 to 400 (as opposed to the 1000 that we did).
